linux上手笔记

linux常用命令

yum使用

  1. yum list 查看安装软件
  2. yum remove xx移除软件

服务器操作

  1. 查看内存使用情况：free -m ，-m代表使用m为单位，可选-g，
  2. 硬盘情况：df -h， -human，以人类易识别方式显示
  3. 负载： w/top
  4. 查看cpu ：cat /proc/cpuinfo
  5. fdisk :格式化磁盘

文件操作

  1. touch 新建文件
  2. rm：删除文件

文件操作
命令 按键 备注
文件尾部开始读 tail filename
完整读取文件 cat filename
查看关键字行 grep -n ‘key’ filename 查看有key内容的行，并显示行数（-n）
统计行数 cat filename | wc -l 显示文件行数,wc:word count

查看压缩文件 tar -t(v)f 查看压缩文件（显示详细信息）
解压缩文件 tar -xf 压缩包

用户管理

功能 命令 备注
添加账户 useradd 需要指定用户参数
设置密码 passwd username 给用户设置密码

useradd命令中的用户参数以及作用

参数 作用
-d 指定用户的家目录（默认为/home/username）
-e 账户的到期时间，格式为YYYY-MM-DD.
-u 指定该用户的默认UID
-g 指定一个初始的用户基本组（必须已存在）
-G 指定一个或多个扩展用户组
-N 不创建与用户同名的基本用户组
-s 指定该用户的默认Shell解释器

防火墙

  1. yum list |grep firewalld ：查看firewalld防火墙软件是否安装
  2. service firewalld stop/start :开关防火墙

提权和文件上传下载操作

提权

  1. 使用root用户输入visudo，在以下行中输入：%用户名 ALL=(ALL) ALL

  2. 普通用户使用sudo 即可执行超管命令

文件上传
linux/mac下
  1. scp实现文件上传与下载 linux或者mac使用此种方式
windows下

  1. 使用xshell的属性》文件传输，使用默认Zmodem（默认）传输

  2. 服务器安装 yum install lrzsz

    • 上传

    xshell 中输入rz，选择文件传输到当前目录

    • 下载

    sz 文件名，选择保存地址
