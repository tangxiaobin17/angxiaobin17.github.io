什么是Git

  • Git：分布式管理系统
  • SVN：集中式管理系统

工具软件

  • Git TortoiseGit 类似于svn的小乌龟
  • VS Code ：版本管理辅助软件

使用方法

主分支

  1. 服务器端创建版本库

  2. 本地克隆服务器工程

  3. 本地修改&提交

  4. 服务器推送

新需求、开分支

  1. 创建dev分支

  2. 开发

  3. 合并分支

  4. 推送分支

分支合并技巧
  • cherry pick :压缩提交（建议使用这种，保证主分支整洁）
  • merge branch：将分支所有提交合并到当前分支
