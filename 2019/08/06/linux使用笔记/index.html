linux自学笔记

查看服务器地址

  1. ifconfig；查看服务器地址

    刚安装的虚拟机可能不能解析主机地址，原因为未连接网络：

    1. ip add 查看网卡地址，记住ens33样式的名字
    2. 查找配置文件

    1. vi ifcfg-ens33

      最底下
      ONBOOT = yes

    2. 修改完毕后
      service network restart

    3. ping www.baidu.com

服务器管理注意事项 20190726

  1. 远程服务器不能关机，否则只能管理人员帮忙开机

  2. 服务器重启前需要终止正在执行的服务；使用“shutdown -r now”重启，正常保存和中止服务器中正在运行的程序，是安全命令

  3. 最好在重启前执行几次 “sync” 命令，这条命令是数据同步命令，可以让暂时保存在内存中的数据同步到硬盘上。

用户、用户权限管理

  1. useradd 选项 用户名 ：创建用户，并创建用户目录
  2. userdel 选项 用户名 ：删除用户，-r 把用户的主目录一起删除。
  3. passwd 选项 用户名 ：修改创建密码

linux虚拟机上传文件

  1. 安装工具：yum install lrzsz -y
  2. 检查是否安装成功：rpm -qa |grep lrzsz
  3. xshell中输入 rz ，对话框中选择上传文件
  4. sz {文件} 弹出选择本地保存文件对话框
